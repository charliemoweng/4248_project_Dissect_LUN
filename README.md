# CS4248 Group 27 Project. 
This is Group 27's NLP Project with LUN Datasets

## Folder Structures
1. datasets -> Contains all the datasets used for the project
2. notebooks -> Contains all the notebooks that was used for the project. Need Google Colab
3. scripts -> Contains the script that augments our datasets.

## Getting Started
To get started, you will need to:-
Local Development
1. Setup an Python 3.10 environment
2. pip install -r requirements.txt

Google Colab
1. Upload the required files to Google Colab to run our codebase.
